# OpenSoho Home Assistant Add-on

OpenSoho is a Small Office Home Office (SOHO) network management tool for OpenWrt devices. This add-on allows you to manage your routers directly from Home Assistant.

## Features

- Manage 2 to 20 OpenWrt devices.
- Simple to deploy and configure.
- Compatible with openwisp-config and openwisp-monitoring.
- Home Assistant Ingress integration.
- Host network support for device discovery.
